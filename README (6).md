# Blinkit Sales Analysis (Tableau Project)

This project analyzes Blinkit's sales data using **Tableau** to uncover insights about product performance, outlet sales, and customer demand patterns.

The dashboard transforms raw retail data into **interactive visualizations** that help identify revenue trends, top-performing products, and outlet performance.

---

## Project Objective

The objective of this project is to demonstrate how **Tableau can be used for data analysis and business intelligence reporting**.

The analysis focuses on:

- Understanding overall sales performance
- Identifying top-selling product categories
- Analyzing outlet performance
- Exploring sales trends across locations
- Supporting data-driven business decisions

---

## Dataset Overview

The dataset includes information related to Blinkit's retail operations.

### Key Columns

| Column | Description |
|------|-------------|
| Item Identifier | Unique product ID |
| Item Type | Product category |
| Item Fat Content | Low fat or regular product |
| Outlet Identifier | Unique outlet ID |
| Outlet Location Type | City tier or location type |
| Outlet Size | Size of the outlet |
| Outlet Type | Type of store |
| Item MRP | Maximum retail price |
| Item Visibility | Product visibility in store |
| Sales | Total sales amount |

---

## Dashboard Features

The Tableau dashboard provides several interactive insights:

### Overall Sales KPI
Displays total sales, average sales, and number of items sold.

### Product Category Analysis
Shows which product categories contribute the most to revenue.

### Outlet Performance
Analyzes sales based on outlet size, type, and location.

### Fat Content Analysis
Compares sales between low-fat and regular products.

### Location-Based Insights
Identifies which city tiers generate the highest revenue.
